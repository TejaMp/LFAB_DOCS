Goldengate setup:
================
Replicating 112 tables from multiple INSIS% schemas to INSISHIST schema.
source : INSIS_GEN_V10.CLAIM_INSPECTIONS - n columns
target : INSISHIST.IGV_CLAIM_INSPECTIONS - n+6 columns

"DB_OP_TYPE" => INSERT, DELETE, UPDATE
"SRC_COMMIT_TS" => commit timstamp
"BFR_AFT_IND" => Before OR After
"GG_CSN"  => Goldengate CSN
"GG_TFILE_SEQNO"  => Trail file number
"GG_TFILE_RBA" => Position in the trail file

MAP INSIS_GEN_V10.CLAIM_INSPECTIONS TARGET INSISHIST.IGV_CLAIM_INSPECTIONS , COLMAP ( USEDEFAULTS, db_op_type = @GETENV ('GGHEADER', 'OPTYPE'), src_commit_ts = @GETENV ('GGHEADER','COMMITTIMESTAMP'), bfr_aft_ind = @GETENV ('GGHEADER','BEFOREAFTERINDICATOR'), GG_CSN = @GETENV ('TRANSACTION','CSN'), GG_TFILE_SEQNO = @COMPUTE(@NUMSTR(@GETENV('RECORD','FILESEQNO'))*1), GG_TFILE_RBA = @GETENV('RECORD','FILERBA'));


-- A job to monitor the missing records
runs form OEM, compares data between source and target table using SCN number and send the diff via email.

Setup in PROD, UPGSIT and very few dev envs.
